This is a Next.js App Router frontend for Venopai Commerce.

## Environment

Copy `.env.local.example` to `.env.local` and set values:

```bash
NEXT_PUBLIC_API_URL=https://api.venopai.com
NEXT_PUBLIC_RAZORPAY_KEY=your_key_here
```

API integrations are mapped to `../API_CONTRACT.md` and only use documented endpoints.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
